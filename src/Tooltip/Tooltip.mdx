# Tooltip <small class="bootstrap-docs">[Bootstrap Tooltip](https://getbootstrap.com/docs/5.3/components/tooltips/)</small>

The `<Tooltip>` component is a user interface element that provides supplementary information when users hover over or interact with an element.

## Controlled

<Canvas withSource="none">
  <Story of={TooltipStories.Controlled} />
</Canvas>

<Source
  dark
  language="html"
  code={`
<script lang="ts">
  import { Button, Tooltip } from '@sveltestrap/sveltestrap';
  let isOpen = false;
</script>

<div class="mt-3">
  <Button id="controlledBtn">You could hover here</Button>
  <Tooltip bind:isOpen placement="right" target="controlledBtn">
    This is a Tooltip controlled externally.
  </Tooltip>
  <hr />
  <label>
    <input type="checkbox" bind:checked={isOpen} />
    Or you can check this to control the Tooltip state.
  </label>
</div>
`} />
